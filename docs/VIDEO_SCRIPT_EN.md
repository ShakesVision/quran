> "Want a distraction-free experience? Tap the expand button for immersive mode — all toolbars disappear."

## Screen Recording Checklist

- [ ] Record on a clean phone (no personal notifications)
- [ ] Use portrait orientation
- [ ] 1080x1920 resolution minimum
- [ ] Record each section separately for easy editing
- [ ] Use smooth, deliberate gestures (slow swipes, visible taps)
- [ ] Ensure good wifi for any API-dependent features (translations)
- [ ] Show dark mode briefly at some point
- [ ] End with the home screen showing the logo
